角色與入口

系統依照帳號角色與部門顯示不同功能。登入後，上方導覽列會只顯示你有權限使用的功能選單。

角色 / 部門 主要入口 適用情境
管理者 / admin 管理介面、系統設定、訂單、發單作業、薪資、員工資料 權限管理、系統維護、跨部門資料調整
行政 / 業務（dept 1） 訂單、發單作業、客戶、估價、派車表單 建單、發單、對客、查單、客戶資料維護
生產（dept 3） 生產 查看工單、處理生產進度與現場關卡
一般員工 / 行動帳號 打卡、請假 / 加班、員工查詢、庫存、繪圖工具 日常查詢、出勤、查看訂單與完工照片
若你看不到某個選單，通常不是系統壞掉，而是目前帳號沒有該功能權限。這時請找管理者確認角色或部門設定。

常用流程

行政常用

新訂單流程

  1. 進入「訂單」建立或編輯資料
  2. 補齊客戶、石材、台面、水槽、爐具與附件
  3. 送出確定單給客戶簽回
  4. 確認後執行發單，系統產生正式訂單號
  5. 系統可接續建立生產工單與後續關聯資料
生產常用

工單處理流程

  1. 進入「生產」查看待處理工單
  2. 依工序確認裁切、水刀、黏合、水磨、套板、驗收
  3. 搭配確定單 PDF 或圖面確認加工內容
  4. 更新狀態後，行政端可同步看到最新進度
現場常用

安裝完工與分享

  1. 員工查詢訂單後打開完工照片視窗
  2. 上傳多張照片或影片
  3. 需要時可替換、刪除或分享給客戶
  4. 系統會同步備份與保存分享紀錄
報價常用

估價單流程

  1. 進入估價頁，選擇廚具估價或工地估價
  2. 填寫項目、單位、工資、圖片與客戶資料
  3. 可存檔、載入、公開分享與列印輸出
  4. 需要時可匯出 Excel 或轉為正式報價單

訂單與發單

訂單列表

  • 可依預交日、狀態、訂單號、客戶、石材、地址排序與篩選
  • 可直接進入編輯、繪圖、確定單等下游作業
  • 狀態包含草稿、待回簽、已確認、生產中、已驗收、完工、已取消

訂單編輯

  • 可維護客戶、石材、台面、水槽、爐具、生產指令與附件
  • 可送出確定單、發單、查看繪圖與 PDF
  • 已發單後可修改訂單號，系統會同步更新關聯資料

確定單與繪圖

  • 支援一字型、L 型、M 型與中島繪圖
  • 可產出生產確定單，作為現場與工廠對圖依據
  • 若 PDF 已封存，可供後續發單與派工直接使用

發單作業

  • 可批次處理訂單發單與列印
  • 可查看哪些訂單已封存 PDF、是否已有發單紀錄
  • 支援依關卡輸出多頁派工用途 PDF

維修單

維修單用來處理售後修繕需求，可新建、編輯、列印，並且關聯原始訂單。當來源訂單資料被調整時，系統會盡量同步關聯欄位，避免維修單找不到來源。

生產與施工

生產流程頁

  • 提供工單列表與生產階段追蹤
  • 可搭配訂單號、客戶與預交資訊快速找單
  • 資料會與訂單端互相對應，減少重工輸入

派車調度 / 派車表單

  • 管理安裝任務、車次與排程表
  • 適合出車前確認地點、順序與待辦事項
  • 與訂單資料串接，可降低手抄錯誤

完工照片管理

  • 支援多檔上傳、替換、刪除與分享
  • 適用格式包含圖片與影片
  • 大檔案會有進度條，失敗時系統會自動重試

今日我的任務

  • 管理者可查看當日個人安裝或調度任務
  • 適合每天早上快速確認自己負責內容

若你是現場使用者，只需要記住兩件事：查訂單時先確認訂單號與地址，上傳完工照片時確認有上傳完成再離開頁面。

估價與客戶

廚具估價單

  • 可輸入石材、加工、安裝、特殊項目與附件圖片
  • 支援檔名管理、公開存檔、載入、刪除與分享
  • 可輸出 Excel、報價單、列印版本
  • 網路不穩時可先暫存本機，恢復後再同步

工地估價單

  • 適合工地現場加工與工資估算
  • 水槽、火爐、側落腳等安裝方式可分項設定
  • 支援期貨提醒、公司印章與圖片比例調整

客戶管理

  • 維護客戶基本資料與常用資訊
  • 可搭配訂單建立流程，降低重複輸入
  • 適合行政 / 業務人員集中維護客戶名單

價格與報價溝通

  • 部分 PDF 可對員工端做價格遮罩，避免非必要曝光
  • 客戶報價可透過公開檔或分享方式提供參考

員工、人資與行政

打卡系統

  • 員工可進行上下班打卡
  • 管理者可搭配規則做遲到、早退與曠職判斷

請假 / 加班

  • 員工可送出請假與加班申請
  • 核准後可串接薪資計算

薪資單

  • 管理者可查看薪資結果與扣款資料
  • 支援外勞固定扣項、未滿月比例計薪與加班費
  • 另有專用薪資說明頁供管理者查閱計算邏輯

員工資料

  • 維護人員基本資料、薪資相關欄位與在職狀態
  • 適合管理者做人資與行政設定

員工查詢

  • 可依訂單號、日期、石材、客戶、地址等條件查詢
  • 可直接查看訂單 PDF、拆料單與完工照片

庫存與工具

  • 庫存頁可查詢石材品牌、顏色與尺寸
  • 繪圖工具頁可輔助現場丈量與圖面確認

管理與系統設定

管理介面

  • 管理帳號角色與部門
  • 調整使用者可見功能
  • 進行資料維護或修復作業

系統設定

  • 可設定部分全域參數，例如差勤規則或 PDF 價格遮罩位置
  • 若員工端看到價格遮罩偏移，通常可由管理者在此微調

車輛管理

  • 維護可派車車輛資料
  • 供派車調度與表單使用

維護提醒

  • 修改正式資料前，先確認是否會影響現場、生產或維修流程
  • 若訂單號已發出去，請用系統內建功能修改，不要手動分散改資料

常見問題

Q1：看不到某個功能，是不是壞了？

通常是角色或部門權限未開啟，請找管理者確認帳號設定。

Q2：訂單號改了，其他地方會不會沒更新？

請在系統內使用改訂單號功能，系統會同步處理主要關聯資料，不要分散手動修改。

Q3：估價單網路斷掉怎麼辦？

估價單支援離線暫存，恢復網路後可再同步到 Firebase。

Q4：完工照片要傳給客戶怎麼做？

到照片管理視窗勾選檔案後，可直接分享圖片，或產生相簿連結給客戶。

Q5：如果只想看照片上傳細節呢？

可以另開 員工完工照片上傳說明，那份文件會更細。
