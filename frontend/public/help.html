員工完工照片 / 影片上傳說明

版本 2026-04-28

👉 怎麼進入上傳頁面

  1. 登入員工專區
  2. 查詢你要上傳的訂單（用訂單號碼或安裝日查詢）
  3. 在查詢結果的「完工照片」欄位，點選 「查看 / 上傳」
  4. 跳出照片管理視窗

📤 上傳照片 / 影片

  1. 點選 「選擇檔案」，可一次選多張照片或影片
  2. 確認檔案清單後，點 「上傳照片」
  3. 等待進度條跑完，顯示「上傳完成」即成功

注意事項

項目 說明
支援格式 所有圖片（JPG、PNG…）、影片（MP4、MOV…）
單檔上限 120 MB — 超過會在選檔時提示，請先壓縮影片
多檔上傳 一次可選多個，系統會依序上傳
網路中斷 系統會自動重試 3 次，不需要手動重傳

🔄 更換照片

  1. 找到要換的照片，點「選擇替換檔案」選好新檔
  2. 點 「更新照片」 完成替換

🗑️ 刪除照片

  1. 找到要刪除的照片
  2. 點 「刪除」，確認後即刪除

📨 分享照片給客戶

  1. 勾選要分享的照片（可點「全選」）
  2. 選擇分享方式：
    • 直接分享圖片 — 開啟手機分享選單，直接傳圖片給客戶（圖片較少時較快）
    • 分享相簿連結（穩定） — 產生一個臨時連結（24 小時有效），更穩定，適合同時分享多張或大影片

🗂️ 查看舊有資料夾照片

如果該訂單過去的照片是直接存在 NAS 資料夾（沒有透過系統上傳），點「查看原有資料夾照片」即可瀏覽舊照片。

❗ 常見錯誤處理

錯誤訊息 原因 解決方式
網路錯誤，無法上傳 網路不穩 切換至 Wi-Fi 後重試
上傳逾時 網路過慢或檔案過大 壓縮影片後重試，或換用 Wi-Fi
檔案過大（目前上限 120MB） 影片超過 120MB 先用手機壓縮影片再上傳
權限不足 帳號權限問題 重新登入，或通知管理者確認帳號身分

📁 NAS 資料夾命名規則（供參考）

上傳成功後，新建立的 NAS 資料夾名稱格式：

年-月-日 訂單號碼 石材型號 安裝人員1 安裝人員2 安裝人員3 +車號

範例：

2026-03-18 27292BKY QJ-1201GN 洪仕謙 林義龍 +ABC-1234
已存在的舊資料夾（含地址）不會被改名，僅新建立時不再包含「安裝地點 / 地址」。

🧾 訂單 PDF 的價格區塊（員工看不到）

員工專區開啟訂單 PDF 時，底部「天稅價 / 合計」整列會被白色矩形遮住，員工不會看到價格資訊，避免外流。

  • 適用對象：員工
  • 不受影響：admin、管理者（仍可看到完整 PDF）
如果你發現遮罩位置偏掉、蓋到了不該蓋的地方（例如蓋到簽名或圖面），請通知管理者調整。

⚙️ 給管理者：調整價格遮罩位置

此區僅 admin / 管理者需要看，員工可略過。

進入「系統設定」頁面，找到「員工檢視訂單 PDF 時的價格遮罩位置」區塊，調整以下 4 個比例值（皆為 0~1，代表佔頁面的比例）：

欄位 說明
X 比例 (左) 矩形左邊離頁面左緣的距離
Y 比例 (下) 矩形下緣離頁面底部的距離
寬度比例 矩形寬度佔頁面寬度的比例
高度比例 矩形高度佔頁面高度的比例

預設值（適用目前的「生產確定單」版型）：

X = 0.20    Y = 0.04    寬 = 0.45    高 = 0.13

調整流程：

  1. 用員工帳號開一張訂單 PDF，看遮罩位置偏移方向
  2. 回到「系統設定」微調對應數值（往左 = X 變小、往上 = Y 變大、加寬 = 寬度變大）
  3. 按「儲存」
  4. 員工端重新整理 PDF 即可看到新位置（若沒變，按 Ctrl+F5 強制重抓）

已知問題：特定 PDF 遮罩失效

少數 PDF（通常由特定軟體匯出）使用 PDF 1.5+ 壓縮格式，系統無法解析，遮罩會自動跳過（員工仍可開啟，但看得到價格）。
Cloud Function log 會出現：price redact failed, serving original

解決方式：

  1. 在電腦用 Chrome 開啟問題 PDF
  2. 按 Ctrl + P，目的地選「另存為 PDF」
  3. 用新存的檔案取代 NAS 上的原始檔案即可
