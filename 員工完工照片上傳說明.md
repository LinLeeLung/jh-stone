# 員工完工照片 / 影片上傳說明

---

## 👉 怎麼進入上傳頁面

1. 登入員工專區
2. 查詢你要上傳的訂單（用訂單號碼或安裝日查詢）
3. 在查詢結果的「完工照片」欄位，點選 **「查看 / 上傳」**
4. 跳出照片管理視窗

---

## 📤 上傳照片 / 影片

1. 點選 **「選擇檔案」**，可一次選多張照片或影片
2. 確認檔案清單後，點 **「上傳照片」**
3. 等待進度條跑完，顯示「上傳完成」即成功

### 注意事項

| 項目     | 說明                                          |
| -------- | --------------------------------------------- |
| 支援格式 | 所有圖片（JPG、PNG…）、影片（MP4、MOV…）      |
| 單檔上限 | **120 MB** — 超過會在選檔時提示，請先壓縮影片 |
| 多檔上傳 | 一次可選多個，系統會依序上傳                  |
| 網路中斷 | 系統會自動重試 3 次，不需要手動重傳           |

---

## 🔄 更換照片

1. 找到要換的照片，點「選擇替換檔案」選好新檔
2. 點 **「更新照片」** 完成替換

---

## 🗑️ 刪除照片

1. 找到要刪除的照片
2. 點 **「刪除」**，確認後即刪除

---

## 📨 分享照片給客戶

1. 勾選要分享的照片（可點「全選」）
2. 選擇分享方式：
   - **直接分享圖片** — 開啟手機分享選單，直接傳圖片給客戶（圖片較少時較快）
   - **分享相簿連結（穩定）** — 產生一個臨時連結（24小時有效），更穩定，適合同時分享多張或大影片

---

## 🗂️ 查看舊有資料夾照片

如果該訂單過去的照片是直接存在 NAS 資料夾（沒有透過系統上傳），點「**查看原有資料夾照片**」即可瀏覽舊照片。

---

## ❗ 常見錯誤處理

| 錯誤訊息                   | 原因               | 解決方式                           |
| -------------------------- | ------------------ | ---------------------------------- |
| 網路錯誤，無法上傳         | 網路不穩           | 切換至 Wi-Fi 後重試                |
| 上傳逾時                   | 網路過慢或檔案過大 | 壓縮影片後重試，或換用 Wi-Fi       |
| 檔案過大（目前上限 120MB） | 影片超過 120MB     | 先用手機壓縮影片再上傳             |
| 權限不足                   | 帳號權限問題       | 重新登入，或通知管理者確認帳號身分 |

## 📁 NAS 資料夾命名規則（供參考）

上傳成功後，NAS 上的資料夾名稱格式：

```
年-月-日 訂單號碼 石材型號 安裝地點 安裝人員1 安裝人員2 安裝人員3 +車號
```

範例：

```
2026-03-18 27292BKY QJ-1201GN 桃園市桃園區朝陽街359號 洪仕謙 林義龍 +ABC-1234
```
